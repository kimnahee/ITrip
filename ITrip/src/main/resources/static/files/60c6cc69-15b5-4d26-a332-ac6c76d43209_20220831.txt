1. 게임 커뮤니티

Steam, Origin, 에픽게임즈 참고

- 개요
취미로 게임을 자주 하는데 하고 싶은 게임이 있어도 같이 할 친구가 없으면 오래 즐기지 못하고 삭제하는 경우가 매우 잦았음
Steam 같은 플랫폼은 접근성이 매우 떨어져 같이 할 사람을 구하기도 매우 힘들고 인터넷 커뮤니티 등지에서 따로 구하는 경우가
허다한데 이를 게시판 등을 통해 실시간으로 함께 할 수 있는 유저들을 구하면 어떨까 싶어서 생각해보게 되었음  
그리고 비슷한 장르의 게임을 추천해주기는 하지만 직접 해 본 사람 기준이 아니어서 신뢰도가 조금 떨어진다고 생각해
유저의 추천을 직접 들을 수 있는 게시판이 있으면 좋겠다고 생각했음

[구매/결제, 라이브러리 기능]
게임 구매
구매하면 해당 계정 라이브러리에 추가
(마이페이지 - 라이브러리 연동)
찜 목록, 장바구니?
게임 리뷰 페이지 등등

[커뮤니티 기능]
1. 게임 카테고리 별 게시판, => 비슷한 장르 게임 추천도 해주면 좋겠는데..
2. 같이 할 사람 구하는 게시판 => 댓글을 작성하고 해당 유저 클릭 : 1대 1 개인 채팅 보내기?
3. 같이 할 사람을 구하고 마음이 잘 맞았다면 그룹 등을 생성할 수 있는 기능 추가
=> 단체로 다음 약속을 정하거나 게임 진행도를 기록할 수 있는 기능 추가(캘린더 API & TODO LIST등을 사용해서..)









2. 고양이(or 반려동물) 동물병원/왕진시스템

- 개요
반려묘를 집에서 동물병원으로 데리고 다니는 일이 매우 힘들어서 생각하게 되었음
(데리고 갈 일정 조율이 쉽지 않은 편이며 직장인의 경우에는 저녁에 동물병원을 가기가 쉽지 않아서)
찾아보니 집으로 방문해서 방문진료를 해주는 동물병원도 가끔 보여서 이러한 서비스를 
쉽게 접근할 수 있도록 도입하면 좋을 것 같다고 생각

- 주요기능

기본적인 회원가입, 로그인, 마이페이지 
 - 집 근처에 있는 병원 검색 및 확인(공공데이터 API - 동물병원 리스트 활용)
   위치 확인
 - (왕진이 가능한 병원의 경우) 따로 서비스를 등록해서 리스트로 확인가능(평가/리뷰/별점 도입해서 서비스가 어떤지 확인할 수 있음)
    왕진 서비스 신청 시 해당 병원에 방문일정 조율(캘린더 사용?)/증상 혹은 방문이유 등을 작성해서 신청
    동물병원 측에서 확인 후 일정 잡기
    이 경우 진료예약비 혹은 진료비 미리 납부(결제 시스템 등록)
 - 캘린더에는 병원이 등록한 일정 확인(미리 등록해둔 예약 등등을 확인)할 수 있도록 < 수정

반려묘 수첩 + 마이페이지
 - 반려묘 등록정보(다수 등록시 리스트로 확인가능)
 - 최근 예방접종일/체중/사상충/기생충/최근 검사일 
 - 일정 확인(병원 진료 예약일, 방문 예정일, 최근 접종일을 표시)
 - 최근 진료일, 반려묘 등록정보, 진료예정일, 백신 접종예상일 등을 확인
 - 기본 커뮤니티 기능(카테고리 추가)

동물병원 리스트 조회 => 시도, 시군구 기준으로 동물병원 리스트를 검색가능
내 주변 동물병원을 찾아야 하니까 지도 API를 활용해서 좀 더 객관적인 위치를 파악가능할수 있으면 좋을 것 같음

우리엔(Woorien 어플)
동물 등록정보/접종일 확인기능은 매우 편리하게 사용했음
그렇지만 등록정보가 누락된 경우도 다수 있었고 어플 자체에서 병원 예약도 가능했으면 좀 더 편했을 것 같았음

기대효과
일상생활이 바쁜 직장인의 경우 보통 주말에만 동물병원을 방문하는데 이러한 왕진 서비스를 통해
반려동물의 진료를 쉽게 받을 수 있음
=> 또한 주말에 동물병원에 가더라도 병원과 일정 충돌(수술같은 경우)로 인해 장시간 대기해야 하는 경우가 있고 병원 일정 확인을 통해 이를 개선할 수 있다고 생각함
집 근처의 동물병원이 맞지 않아서 다른 병원을 찾아봐야 할 경우, 집 근처의 다른 동물병원을 쉽게 찾아볼 수 있다
(근처에 있는 동물병원을 확인하기 힘들어서 매번 같은 병원에만 갔던 적이 있었기 때문에)
